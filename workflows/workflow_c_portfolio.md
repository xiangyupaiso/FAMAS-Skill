# Workflow C: 组合持仓诊断 (Portfolio Diagnosis)

## 工作流概述

对用户当前持有的多只基金进行组合级风险诊断，分析集中度、相关性，并给出再平衡建议。

## 执行流程

```
用户输入: 持仓列表(基金代码+比例)
    |
    v
+----------------------+
| portfolio_doctor     | --> 组合级风险扫描
+-----------+----------+
            |
+-----------v-----------+
| cost_analyzer        | --> 组合总持有成本计算
| fund_comparator      | --> 持仓基金间重合度分析
+-----------------------+
```

## 详细步骤

### Step 1: 组合风险扫描 (portfolio_doctor)

**输入**: 用户持仓列表(基金代码+持有比例)

**执行**:
- 分析组合结构（股债配比、行业分布、风格分布）
- 评估集中度风险
- 分析风格偏离度
- 评估隐性相关性
- 生成再平衡建议

**输出**:
```json
{
  "asset_allocation": {
    "stock_ratio": "股票占比",
    "bond_ratio": "债券占比"
  },
  "concentration_risk": {
    "top3_sector_ratio": "前三大行业占比",
    "risk_level": "风险等级"
  },
  "style_deviation": {
    "value_growth_deviation": "价值/成长偏离",
    "deviation_risk": "偏离风险"
  },
  "correlation_analysis": {
    "hidden_concentration": ["隐性集中识别"],
    "tail_correlation": "尾部相关性"
  },
  "rebalancing": {
    "current_issues": ["当前问题"],
    "adjustment_plan": ["调整方案"]
  }
}
```

### Step 2: 成本与重合度分析

**输入**: 持仓列表 + Step 1输出

**并行执行**:

**cost_analyzer**:
- 计算组合总持有成本
- 分析费率侵蚀
- 评估成本优化空间

**fund_comparator**:
- 分析持仓基金间重合度
- 识别同质化持仓
- 评估分散效果

**输出**:
```json
{
  "portfolio_cost": {
    "total_tcr": "组合总持有成本",
    "fee_erosion": "费率侵蚀"
  },
  "overlap_analysis": {
    "overlap_matrix": "重合度矩阵",
    "homogeneity_risk": "同质化风险"
  }
}
```

## 使用示例

```
用户: 帮我诊断一下我的持仓: 020712(30%), 005911(25%), 161725(20%), 000083(25%)

系统:
1. portfolio_doctor 扫描组合级风险
2. cost_analyzer 计算组合总持有成本
3. fund_comparator 分析持仓基金间重合度

输出: 组合风险画像 + 集中度热力图 + 再平衡建议 + 隐性相关性报告
```
